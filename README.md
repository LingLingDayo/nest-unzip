# NestUnzip

NestUnzip 是一款极简、高颜值的 Tauri 桌面应用，专为深度嵌套、批量解包及多密码尝试场景而设计。

---

## ✨ 核心特性

*   **自动路径检测**：启动时自动搜寻系统中已安装的 `7-Zip` 或 `Bandizip`，支持通过设置指定其**安装目录**与首选解压引擎，无须手动指定复杂的 exe 绝对路径。
*   **深度嵌套解压**：一键解包“压缩包套压缩包”的嵌套架构。解压完毕后，所有中间过渡的子压缩文件将根据配置**移入回收站**或**直接物理删除**，仅保留最终解压的非压缩文件，省时又省力。
*   **异常回滚机制**：如果解压中途失败（如密码错、损坏等），软件会自动触发回滚，将已解压生成的所有临时脏文件、半截文件夹统统**移入回收站**或**直接物理删除**（取决于设置），避免污染工作目录。
*   **智能密码交互**：支持配置全局密码列表和任务专属密码列表。若内置密码池耗尽，会**自动弹窗提示用户手动输入**；解密成功后，该密码将**自动保存**至该任务的专属密码中，方便后续解包。
*   **批量选择与防覆盖**：支持批量拖入压缩包、**勾选部分任务进行选择性解压**，并支持每个任务自定义目标路径与专属密码。若解包目标目录重名，会自动递增后缀（如 `(1)`）进行防覆盖处理。
*   **实时控制台终端**：配备底层事件追踪日志，详细展示各层级解包进度、目录结构、密码尝试及出错报告。

## 🚀 简易使用教程

### 1. 配置解压引擎
首次使用或检测失败时，点击右上角的 **设置** 按钮：
*   指定 `7-Zip` 或 `Bandizip` 的安装目录（通常为 `C:\Program Files\7-Zip` 或 `C:\Program Files\Bandizip`，留空则由系统自动检测）。
*   配置首选解压引擎（默认使用 7-Zip）。
*   在“通用解包设置”中填入一些你经常使用的**全局默认密码列表**（每行一个）。
*   配置“完全删除中间产物（不移入回收站）”选项，决定临时文件和子包是被永久物理删除还是移入回收站。
*   可开启“解压完成后自动打开目标文件夹”的选项。

### 2. 导入压缩包任务
支持两种方式：
*   **拖放导入**：直接将压缩包拖入软件的任何区域。
*   **点击导入**：点击主界面的“添加压缩包”虚线框或上方工具栏的“添加压缩包”按钮。

### 3. 微调任务配置
每个导入的文件会作为一个卡片显示：
*   **自定义目标路径**：默认在同级生成同名目录（若同名文件夹已存在，将自动加上 `(1)`、`(2)` 等后缀）。你可以点击卡片上的 **更改** 按钮，选择任意其他文件夹。
*   **选择任务**：可以通过点击卡片左侧进行勾选或取消勾选，以支持选择性批量解包。
*   **解压密码**：如果该包有专属密码，可在卡片输入框中直接填入（若有多个密码，可用空格隔开，程序将优先于全局密码依次尝试）。若密码池穷尽，中途会弹出框请求输入。

### 4. 执行解压与查看
点击工具栏的 **开始批量解压** 或 **解压已选压缩包**：
*   控制台会依次高亮当前任务，并输出实时解压进度条。
*   点击任务卡片可展开/收起该任务对应的详细提取日志。
*   底部控制台会输出全局任务报告。

## 🛠️ 开发者指南

### 本地依赖安装
建议使用 `pnpm`：
```powershell
pnpm install --ignore-scripts
```

### 启动本地开发调试
```powershell
pnpm tauri dev
```

### 生产版本打包
```powershell
pnpm tauri build
```
